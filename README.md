# Scrimba Advanced JavaScript Course

Complete solutions and notes for Scrimba's Advanced JavaScript Topics course.

## 📚 Course Overview

This repository contains comprehensive solutions to exercises and challenges from the Scrimba Advanced JavaScript course. Each file covers a specific advanced JavaScript concept with practical examples and implementations.

## 📂 Course Contents

### 1. Ternary Operators
- **[01TernaryOperator.js](01TernaryOperator.js)** - Introduction to ternary operators
- **[02TernaryOperatorComplex.js](02TernaryOperatorComplex.js)** - Complex and nested ternary operations
- **[03TernaryChallange.js](03TernaryChallange.js)** - Practice challenges

### 2. Control Flow
- **[04SwitchCase.js](04SwitchCase.js)** - Switch case statements and best practices

### 3. Destructuring
- **[05ObjectDestructuring.js](05ObjectDestructuring.js)** - Object destructuring fundamentals
- **[06ObjectDestructuringChallange.js](06ObjectDestructuringChallange.js)** - Destructuring practice exercises

### 4. Timing Functions
- **[07setTimeout.js](07setTimeout.js)** - Basic setTimeout usage
- **[08setTimeOutwithParams.js](08setTimeOutwithParams.js)** - Passing parameters to setTimeout
- **[09setTimeoutwithParamChallange.js](09setTimeoutwithParamChallange.js)** - setTimeout challenges
- **[10setInterval.js](10setInterval.js)** - setInterval fundamentals
- **[11setIntervalwithParams.js](11setIntervalwithParams.js)** - setInterval with parameters

### 5. Built-in Constructors
- **[12DateConstructor.js](12DateConstructor.js)** - Working with Date objects
- **[13ErrorConstructor.js](13ErrorConstructor.js)** - Error handling and custom errors

### 6. Operators & Number Handling
- **[14PreIncrement.js](14PreIncrement.js)** - Pre and post increment/decrement operators
- **[15NumericSeprator.js](15NumericSeprator.js)** - Numeric separators for readability
- **[16BigInt.js](16BigInt.js)** - Working with BigInt for large integers

### 7. Advanced Concepts
- **[17Hoisting.js](17Hoisting.js)** - Understanding JavaScript hoisting

## 🎯 Learning Objectives

By working through this course, you'll master:

- ✅ Conditional logic with ternary operators and switch statements
- ✅ Modern JavaScript destructuring patterns
- ✅ Asynchronous timing with setTimeout and setInterval
- ✅ Built-in JavaScript constructors and objects
- ✅ Advanced number handling and operators
- ✅ JavaScript execution context and hoisting

## 🚀 Getting Started

1. Clone this repository:
```bash
git clone https://github.com/nishisurti2608/Scrimba-Advance-Javascript.git
```

2. Navigate to the project directory:
```bash
cd Scrimba-Advance-Javascript
```

3. Open any JavaScript file in your preferred editor or run them with Node.js:
```bash
node 01TernaryOperator.js
```

## 💻 Prerequisites

- Basic understanding of JavaScript fundamentals
- Node.js installed (optional, for running files locally)
- A code editor (VS Code, Sublime Text, etc.)
